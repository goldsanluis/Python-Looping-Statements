# Python Looping Statements

## Description
This repository contains Python exercises focused on looping statements, including prime number checking, array operations, and pattern printing such as hollow squares and inverted triangles. It demonstrates basic problem-solving and programming logic.

---

## Objectives
- Understand the use of looping statements in Python
- Apply `for` and `while` loops in solving problems
- Develop logical thinking through programming exercises
- Create patterns and perform computations using loops

---

## Activities Included
- Prime Number Checker  
- Array Cube Computation  
- Hollow Square Pattern  
- Inverted Triangle Pattern  

---

## Technologies Used
- Python 3
- Jupyter Notebook (.ipynb)
